===========================
Sale Order Line Min. Margin
===========================

This module allows to show a warning message when the profit margin in sale order lines are below a given rate set in the company.


Installation
============

To install this module, you need to:

#. Only install


Configuration
=============

To configure this module, you need to:

#. Go to Sales > Configuration > Settings > Pricing and activate the "Apply Sale Order Line Min. Margin" option when the sale order line margin needs to be equal or greater than a given value.
